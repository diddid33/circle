이 웹사이트는 HTML의 태그와 쿼리를 알파벳 순서로 정리한 가이드입니다.

각 알파벳은 특정 기능을 담고 있으며, 마우스를 올리면 돋보기에 알파벳이 뜹니다.

드래그하거나 아래에 있는 네비게이션 바에서 알파벳을 클릭하여 구경할 수 있습니다.

마우스를 움직여 웹사이트 속에 숨겨진 알파벳을 찾아보세요!

이미지(또는 기타 콘텐츠)를 포함하는 컨테이너 태그이다. <figcation> 함께 사용된다.

M
<mark>
형광펜 효과로 텍스트를 강조하며, CSS로 배경색·테두리·모서리 등을 조절할 수 있습니다.
N
<input type="number">
숫자 입력 필드로, 사용자가 숫자를 입력하거나 ↑↓ 버튼으로 값을 조절할 수 있습니다.
O
oppacity
투명도를 조절하는 CSS 속성으로,0은 완전 투명, 1은 완전 불투명을 의미한다.
P
position

셀렉트(Select) 태그로, 드롭다운 메뉴를 만들 때 사용한다.
